# Daily Routine Workflow

## Morning (5-10 min)

1. **Review yesterday's daily note** (if exists)
2. **Set today's intentions**
   - Use `/daily-review` or create note from template
   - Identify top 3 priorities
3. **Quick inbox scan**
   - Anything urgent from `0. Inbox`?

## During the Day

1. **Capture to Inbox**
   - Quick notes â†’ `0. Inbox`
   - Don't organize in the moment
   - Trust the system

2. **Work in Projects**
   - Update project notes as you work
   - Link discoveries to related notes

3. **Thinking Sessions**
   - Use `/thinking-partner` for complex problems
   - Document insights as they emerge

## Evening (5-10 min)

1. **Run Daily Review**
   - Use `/daily-review` command
   - Capture what happened
   - Note insights and learnings

2. **Quick Inbox Process**
   - Move obvious items to proper locations
   - Use `/inbox-processor` if backlog builds

3. **Set Tomorrow's Focus**
   - What's the one thing that matters most?

## Weekly (30-60 min)

1. **Weekly Synthesis**
   - Use `/weekly-synthesis` command
   - Sunday evening or Monday morning

2. **Full Inbox Process**
   - Clear everything or decide to delete

3. **Project Review**
   - Check all active projects
   - Archive completed work

4. **System Maintenance**
   - Any new MOCs needed?
   - Tags getting messy?
