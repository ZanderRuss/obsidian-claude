# Workflow Support

Three helpful agents for planning, research, and improving how Claude works for you.

## What's Here

| Agent | What It Does |
|-------|--------------|
| **prompt-engineer** | Makes Claude respond better by improving prompts |
| **search-specialist** | Finds and verifies information from the web |
| **task-decomposition-expert** | Breaks big projects into manageable steps |

---

## prompt-engineer

**Helps Claude communicate more effectively**

Use when you want to:
- Improve how a skill or command responds
- Create better system instructions
- Make Claude's outputs more consistent

What you get:
- Complete prompt text (never just a description)
- Explanation of why it works
- Examples of expected outputs

---

## search-specialist

**Thorough web research and fact-checking**

Use when you want to:
- Research a topic deeply
- Find information from multiple sources
- Verify facts and claims
- Compare different perspectives

What you get:
- Information from multiple sources
- Source verification
- Synthesized findings

---

## task-decomposition-expert

**Breaks complex projects into clear steps**

Use when you want to:
- Start a big project and don't know where to begin
- Plan a multi-step workflow
- Decide between different approaches

What you get:
- Step-by-step breakdown
- Recommended tools for each step
- Dependencies between tasks
- Risk assessment
